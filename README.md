# Accessible Digital Learning Platform
This project is a collaboration between Worcester Polytechnic Institute (WPI) and the Inclusive Tech Lab Project, ran by Videnscenter om Handicap (VOH) and Microsoft.

## Introduction
This project will serve to fulfill the Interactive Qualifying Project for WPI students: Daniel Kim, Matthew Nickerson, Brendon Peters, Patrick Tirch, and Ethan Uzdarwin.

## Project Goal & Purpose
The primary goal of this project is to develop an accessible, educationally motivated website prototype utilizing feedback from the staff and users at the Inclusive Tech Lab. The broader purpose is to assist Videnscenter Om Handicap in improving the digital experiences of people with disabilities and setting reformed guidelines for web accessibility. 

This application will serve as a dedicated learning platform for individuals who are interested in exploring technology, but face barriers related to accessibility, confidence, or limited support.

## Key Objectives
* **Identify Needs:** Identify the barriers and needs of persons with disabilities regarding accessible technology and education.
* **Assess Tools:** Assess existing digital learning platforms and accessibility tools based on criteria found in the research.
* **Develop Prototype:** Develop an educational web application prototype that meets the accessibility needs of our user base.
* **Iterative Evaluation:** Evaluate the prototype through active user studies, measuring user responses to iteratively improve the application.

## Accessibility Standards & Methodology
The development of this platform is driven by inclusive design principles and effective educational theories:
* **POUR Principles:** Ensuring all features are Perceivable, Operable, Understandable, and Robust.
* **WCAG 2.1 Compliance:** Incorporating specific guidelines such as text alternatives, keyboard accessibility, navigability, readability, and compatibility.
* **Cognitive Load Management:** Designing instructional content to align with human memory limits, avoiding overwhelming users to ensure maximum comprehension.
* **Co-Creation & Testing:** Working alongside users in a process called co-creation by holding testing sessions with persons with disabilities to evaluate interaction and implement feedback.
